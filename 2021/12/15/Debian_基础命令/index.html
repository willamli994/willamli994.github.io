Debian常用命令

发表于2021-12-15|更新于2021-12-20
|字数总计:897|阅读时长:3分钟|阅读量:

参考博客

有很多理由可以选择 Debian 作为您的操作系统——作为用户、作为开发者，甚至是在企业环境中。大多数用户称赞它的稳定性，以及软件包和发行版的平滑的升级过程。Debian 也被软件和硬件开发人员广泛使用，因为它能运行在众多架构和设备上，提供了一个公开的缺陷跟踪系统，以及面向开发人员的其他工具。如果您在专业环境中使用 Debian，您还可以享受到诸如 LTS 版本和云映像带来的额外好处。

系统有很多命令，下面列出常用命令，如有其他需求点击此处，*表示重要的命令

1、文件/文件夹管理常用命令:

            ls 列出当前目录文件 （不包括隐藏文件）
            ls -a  列出当前目录文件 （包括隐藏文件）
            ls -l 列出当前目录下文件的详细信息  *
            
            cd ..  回当前目录的上一级目录
            cd -    回上一次所在的目录
            cd /home   进入系统根目录(进去其他文件直接打路径) *
            mkdir /test   在根目录创建一个名为test的目录
            rmdir /test    删除一个空目录
            rm -rf   删除一个非空目录下的所有文件
            
            mv /test /home/test   把根目录的test文件移到home目录下的test
            mv /test /test1     在根目录下把test改名为test1 (test有移动文件功能也有改名功能)

2、系统管理常用命令:

            fdisk  -l  常看系统分区信息
            fdisk /dev/sdb  为一块硬盘进行分区
            chown root /home   把/home的属主改成root用户
            chgrp root /home   把/home的属组改成root组
            
            useradd test  创建名为test的新用户
            groupadd Test  创建名为Test的新组
            passwd test   为test用户创建密码
            
            service [servername] start/stop/restart 系统控制操作
            #service networking start   开启网络服务
            #service networking stop     关闭网络服务
            #service networking restart   重启网络服务
            
            df -h    查看挂载信息
            free -m  查看内存使用情况
            ps -A    查看进程
            kill 进程号 (ps -A中的第一列数字)
            kill -9 进程号 强制杀死一个进程
            
            reboot  重启系统
            poweroff  关闭系统
            
            设置主机名称(hostname)和fqdn
            vim /etc/homename    修改主机名
            vim /etc/hosts        修改fqdn

3、打包/解压

            tar -c 创建包 -x 解压包 -v 显示命令过程 -z 代表压缩包
            tar -cvf test.tar /home/test   把/home/test目录打包  *
            tar -zcvf test.tar.gz /test    把目录打包并压缩    *
            tar -zxvf test.tar.gz          压缩包的文件解压恢复
            tar -jxvf test.tar.bz2         解压缩

4、apt命令

            package=包(你要搜索的服务，例如Apache就写apache2)
            apt-cache search package   搜索包
            apt-cache show package   获取包的相关信息,如说明、大小、版本
            apt-get install package    安装包
            apt-get remove package     删除包
            apt-get update             更新源
            apt-get clean              清理所有软件缓存
            
            apt-cdrom add              增加光盘源

5、网络

            ip add         查看IP地址信息
            netstat -rn sudo route -n      查看路由信息
            
            让网络支持nat
            sudo echo 1 > /proc/sys/net/ipv4/ip_forward 
            sudo iptables -t nat -I POSTROUTING -j MASQUERADE 
            
            添加和删除路由
            sudo route add -net 192.168.0.0 netmask 255.255.255.0 gw 172.16.0.1 
            sudo route del -net 192.168.0.0 netmask 255.255.255.0 gw 172.16.0.1
            
            远程连接
            ssh -p 22 test@172.16.15.*        22端口远程连接到服务器
            
            ssh传输文件
            #scp -rp /path/filenameusername@remoteIP:/path  把本地文件复制到服务器
            ##scp -rpusername@remoteIP:/path/filename/path  把远程文件下载到本地
