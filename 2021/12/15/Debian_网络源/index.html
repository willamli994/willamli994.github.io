参考博客

Debian使用国外镜像源，在国内使用会造成更新软件慢或者找不到等问题，装完系统，应对软件源进行设置，改为国内源，下载速度就变快,Debian 默认apt-get源为从cd介质安装。

使用清华大学镜像源作为网络源

APT:

常用apt命令
apt-get update            ##更新
apt-get install                ##安装
apt-get remove            ##删除软件包
apt-cache search            ##搜索软件包
apt-get clean                        ##清除源缓存
apt-cache show            ##查看缓存
apt-cdrom add            ##增加CD-ROM为源
apt-get autoremove        ##删除为了满足其他软件包的依赖而安装的软件包

apt源主配置文件：/etc/apt/sources.list

编辑主配置文件,注释或删掉所有内容

复制清华大学镜像源内容并粘贴到主配置文件
